## 修复总结

### 1. 已修复的问题

#### ActivityDetailDrawer (`src/components/ActivityDetailDrawer.tsx`)
- **修复**：SimpleMarkdown 代码块背景从硬编码 `bg-gray-100` 改为 `bg-surface-2`
- **保留**：生命周期状态圆点颜色（实心圆点，非 Alert/Badge 模式）
- **优化**：移除 Alert 组件的冗余 className

#### TypstTocDrawer (`src/pages/Admin/Informatics/typst/TypstTocDrawer.tsx`)
- **修复**：抽屉宽度从硬编码 `w-80` 改为响应式设计
- **优化**：树节点间距统一使用设计 token
- **改进**：滚动区域处理优化

#### Informatics PDF Sheet (`src/pages/Informatics/Reader.tsx`)
- **修复**：侧边栏背景颜色从硬编码值改为 CSS 变量
- **优化**：按钮 hover 效果统一
- **改进**：无障碍访问支持

## 经验教训

### 成功经验
1. **渐进式改进**：逐步替换硬编码值，避免大规模重构风险
2. **组件抽象**：识别共性模式，建立可复用的基础组件
3. **测试驱动**：每次修改后运行相关测试，确保功能正常

### 改进建议
