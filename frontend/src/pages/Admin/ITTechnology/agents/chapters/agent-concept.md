# Agent 概念：从"答题器"到"自主行动者"

你有没有想过，如果手机里的 Siri 不只是回答"明天天气怎么样"，而是能**自己动手**帮你查天气、订机票、发消息、提醒你带伞？

现在的 AI 就像一个**百科全书**——你问它答，仅此而已。但 **AI Agent（智能体）** 不一样：它不只说话，还会**做事**。

## 一、AI Agent 是什么？

### 一个简单的对比

**普通 AI（聊天机器人）**：
> 你："帮我规划周末的生日派对。"
> AI："你可以准备蛋糕、气球，邀请朋友来家里。"

**AI Agent（智能体）**：
> 你："帮我规划周末的生日派对。"
> Agent：
> 1. 先查你的日历，找到你周末空闲的时间
> 2. 搜索附近的蛋糕店，比较价格和评价
> 3. 在班级群里发消息统计谁能来
> 4. 帮你在网上下单订蛋糕
> 5. 生成一份购物清单发到你手机上

看出区别了吗？普通 AI 只会给你建议，Agent **真的去做了**。

### 原神比喻

如果你玩过《原神》，理解 Agent 就太容易了：

- **普通 AI（Chatbot）**就像游戏里的 NPC——你点击它，它说几句固定的台词，永远站在同一个地方，不会主动做任何事。
- **AI Agent**就像你操控的主角——它会自己探索地图、看到敌人自动战斗、收集路上的资源、血量低了就吃药。它能**感知**周围有什么（看到敌人靠近），**思考**下一步做什么（用哪个技能打），然后**行动**（释放技能）。

Agent 的工作方式就是一个循环：

```
观察周围 → 思考对策 → 动手执行 → 观察结果 → 继续思考 → ...
```

就像一个玩家：先看屏幕上有什么，想想该干什么，然后按键操作，再根据结果调整下一步。

### Agent 的四件装备

一个完整的 AI Agent 就像你玩游戏时的角色，需要四样东西：

| 组件 | 游戏类比 | 现实类比 | 作用 |
|------|----------|----------|------|
| **大脑（LLM）** | 你的大脑 | 你的大脑 | 理解问题、做决策 |
| **工具系统** | 角色的武器和技能 | 你的手和脚 | 搜索、计算、发消息、操作 App |
| **记忆系统** | 游戏存档 | 你的笔记本 | 记住刚才聊了什么、你的偏好 |
| **规划模块** | 任务列表 | 你的计划表 | 把大任务拆成小步骤 |

**一句话总结**：普通 AI 只会"说话"，Agent 会"做事"。

### Agent 的自主等级

就像自动驾驶有 L1 到 L5，Agent 也有不同级别：

| 级别 | 名称 | 游戏类比 |
|------|------|----------|
| L0 | 纯聊天 AI | 只会说话的 NPC |
| L1 | 用一次工具 | 你让 AI 搜索，它搜完告诉你 |
| L2 | 简单 Agent | AI 自己搜索、整理、回答 |
| L3 | 自主 Agent | AI 自己拆解复杂任务，想办法完成 |
| L4 | 完全自主 | 科幻电影里的 AI 管家 |

目前（2026 年）的 Agent 大多在 L2-L3 之间——已经很厉害，但还不是科幻电影里那种。

## 🎮 类比理解

### Agent 就像你在原神里操控的角色

```
原神角色:
  看到丘丘人 → 判断距离和数量 → 选择技能 → 释放技能 → 看打没打死 → 继续或换策略
  
AI Agent:
  收到用户问题 → 分析需要什么信息 → 选择工具 → 执行工具 → 看结果够不够 → 继续或回答
```

### Agent 和普通 AI 处理同一个任务

```
普通 AI："帮我查机票"
  → "建议你去携程查一下"（只说不做）

Agent："帮我查周五去上海的机票"
  → 第1步思考：用户要周五去上海，先看看天气
  → 第2步行动：调用天气工具 → 周五上海晴天
  → 第3步思考：天气没问题，查航班
  → 第4步行动：调用机票工具 → 返回3个航班
  → 第5步回答："周五有3个航班：国航14:00 600元、东航16:00 550元、海航18:00 500元。要帮你订吗？"
```

## 🔬 动手实验：最简单的 Agent

```python
# 这是你能写的最简单的 Agent！
# 它会观察、思考、然后行动

class SimpleAgent:
    def __init__(self, name):
        self.name = name
        self.memory = []  # 记忆

    def observe(self, message):
        """Agent 看到/听到什么"""
        print(f"{self.name} 收到了: {message}")
        self.memory.append(message)
        return message

    def think(self, message):
        """Agent 思考要做什么"""
        if "天气" in message:
            return "查天气"
        elif "作业" in message:
            return "查作业"
        elif "时间" in message:
            return "报时间"
        else:
            return "我不知道，直接回答吧"

    def act(self, decision):
        """Agent 执行动作"""
        print(f"{self.name} 决定: {decision}")
        tools = {
            "查天气": "今天晴天，26度，适合出门！",
            "查作业": "数学：P45 3-7题，英语：背Unit5单词",
            "报时间": "现在是下午3点25分",
            "直接回答": "我不太确定，你能再说清楚一点吗？"
        }
        return tools.get(decision, "我做不到这个...")

    def run(self, user_message):
        """完整的一次交互"""
        self.observe(user_message)
        decision = self.think(user_message)
        result = self.act(decision)
        print(f"{self.name} 回答: {result}")
        print("---")
        return result


# 来试试你的 Agent！
agent = SimpleAgent("小A")
agent.run("今天天气怎么样？")
agent.run("今天有什么作业？")
agent.run("现在几点了？")
agent.run("你是男生还是女生？")
```

### 思考题

1. Agent 的"大脑"（`think` 函数）有什么局限？
2. 试着给 Agent 加一个"查日历"的工具。你需要改哪些地方？
3. Agent 的 `memory` 列表有什么用？如果在第三次提问时想用到第一次的内容，怎么做？
4. **挑战题**：让 Agent 在不知道答案时，自动"上网搜索"（用 `print("正在搜索...")` 假装一下）。
